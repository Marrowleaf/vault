# 🏦 VAULT — Finance Command Center

A high-performance, local-first finance and subscription tracker. No SaaS bloat, no cloud dependencies — all data stays in your browser.

**🔗 Live:** [vault.marrowleaf.dev](https://marrowleaf.github.io/vault)

## Features

- **💳 Subscription Tracker** — Track recurring subs, auto-calculate monthly/yearly costs, overdue alerts
- **💰 Transaction Log** — Manual entry + CSV import, search/filter, running totals
- **📊 Budget Planner** — Set monthly budgets per category, track actual vs budgeted spend
- **🏦 Net Worth** — Track assets vs liabilities over time with charts
- **📈 Dashboard** — Full overview with spending trends, upcoming bills, budget status

## Tech

- Pure HTML/CSS/JS — no build step, no framework
- IndexedDB for client-side storage
- Chart.js for visualizations
- Dark theme
- Works offline
- 100% free — no paid APIs or services

## Privacy

All data is stored locally in your browser's IndexedDB. Nothing is sent to any server. Ever.

## Deploy

Just serve the `index.html`, `style.css`, and `app.js` files. Works on GitHub Pages, Netlify, or any static host.
